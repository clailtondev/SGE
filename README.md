
# SGE 

O SGE é um projeto de estudo e desenvolvimento em Python utilizando o framework Django. O objetivo é criar um sistema básico para gerenciamento de cadastros de entidades escolares.

## Funcionalidades Atuais

O sistema possui um módulo inicial de Cadastro de Pessoas (Pessoa), implementando todas as operações básicas do CRUD (Criação, Leitura, Atualização e Exclusão) via interface web.

- **Modelo de Dados:** `Pessoa` (campos: nome, e-mail, celular).
- **CRUD Completo:** Views e templates específicos para listagem, criação, edição e exclusão de registros.
- **Admin:** Interface de administração básica configurada.
